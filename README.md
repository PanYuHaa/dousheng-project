# dousheng-demo

## 功能说明

接口功能不完善，已完成接口如下:
基础接口:
1. 视频流接口
2. 用户注册
3. 发布列表

* 用户登录数据保存在内存中，单次运行过程中有效
* 视频上传后会保存到本地 public 目录中，访问时用 127.0.0.1:8080/static/video/video_name 即可
